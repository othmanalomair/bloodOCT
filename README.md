# دفتر الراوي

تطبيق ويب عربي، مصمم للموبايل، يساعد الراوي على إدارة لعبة Blood on the Clocktower.

## الموجود حالياً

- السكربتات الأساسية: Trouble Brewing وBad Moon Rising وSects & Violets
- توزيع أدوار بحسب عدد اللاعبين (5–15)
- دائرة الجلوس وحالات الحي والميت
- ترشيح وتصويت مع احتساب الأغلبية من الأحياء
- ترشيح نهاري مضبوط: الحي يرشّح مرة، وكل لاعب يُرشّح مرة واحدة في اليوم
- منصة إعدام تتحدث حسب أعلى الأصوات، مع معالجة التعادل وتنفيذ الإعدام عند إنهاء اليوم
- صوت واحد لكل لاعب ميت، ويُستهلك عند تسجيل التصويت
- ترتيب الليلة الأولى والليالي التالية بحسب السكربت
- سجل الترشيحات، ملاحظات الراوي، وتغيير الدور يدوياً
- حفظ تلقائي محلي، مع خيار الاحتفاظ بالأسماء أو مسحها عند بدء لعبة جديدة
- Manifest وأيقونة SVG للتثبيت كتطبيق ويب على الهاتف
- رموز الشخصيات الرسمية ودليل قدرات كامل للسكربتات الثلاثة
- شرح عربي لقدرات جميع شخصيات السكربتات الأساسية مع إبقاء الأسماء بالإنجليزية
- دليل عام قابل للمشاركة على `/guide` بدون أسماء اللاعبين أو بيانات الجلسة
- تعليمات الراوي لكل شخصية ضمن ترتيب الليل

## التشغيل

```bash
npm install
npm run dev
```

للبناء:

```bash
npm run build
```

لتحديث بيانات ورموز الشخصيات من مورد BOTC الرسمي:

```bash
npm run sync:botc
```
